2. Text Styles

You can style text using the following:

  • Bold: **Bold** → Bold
  • Italic: *Italic* → Italic
  • Bold and Italic: ***Bold and Italic*** → Bold and Italic
  • Strikethrough: ~~Strikethrough~~ → Strikethrough

3. Lists

3.1 Ordered List

Output:

  1. Item 1
  2. Item 2
    1. Subitem 2.1
    2. Subitem 2.2
  3. Item 3

3.2 Unordered List

Output:

  • Item A
  • Item B
    • Subitem B.1
    • Subitem B.2
  • Item C

4. Links

You can create links as follows:

  • Inline link: [Google](https://www.google.com)
  • Reference link: [GitHub][github-link]

  • Inline link: Google
  • Reference link: GitHub

11. Math (if enabled)

If you have MathJax enabled, you can write inline math: $E=mc^2$, or block math:

```
$$
\int_a^b f(x)dx = F(b) - F(a)
$$
```

Output:

$$
\int_a^b f(x)dx = F(b) - F(a)
$$

That’s the end of this Markdown guide! Happy documenting!

如何使用

  1. 将以上内容保存为 .md 文件（如 markdown-guide.md）。
  2. 使用任何支持 Markdown 的编辑器（如 VSCode、Typora）打开，查看渲染效果。
  3. 如果在 Hexo 中使用，可以直接作为文章发布。
